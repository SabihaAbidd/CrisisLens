# CrisisLens

A Python project for crisis analysis.

## Setup Instructions

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Configure Environment:
   Fill in `GOOGLE_API_KEY` in `.env` (get it free from aistudio.google.com).

3. Add Demo Assets:
   Put a test image in `demo_assets/` called `test_scene.jpg`.

4. Verify Setup:
   Run:
   ```bash
   python main.py
   ```

5. Launch App:
   Run:
   ```bash
   streamlit run ui/app.py
   ```
